## 🌐 Live Demo

🔗 https://youtube-clone-flax-alpha.vercel.app/

---

## 📌 Overview

This project is a **YouTube Clone** built using modern frontend technologies.
It replicates core UI features of YouTube such as video browsing, search, and responsive layout.

The application consumes **YouTube Fake APIs** to fetch and display video data dynamically, providing a real-world simulation of a video streaming platform.

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash id="clone-repo"
git clone https://github.com/kranthi7581/youtube-clone.git
```

### 2️⃣ Navigate to project folder

```bash id="cd-folder"
cd youtube-clone
```

### 3️⃣ Install dependencies

```bash id="install-deps"
npm install
```

### 4️⃣ Run development server

```bash id="run-dev"
npm run dev
```

### 5️⃣ Build for production

```bash id="build-prod"
npm run build
```
